# 🔴 Push 訓練指南 (胸部、肩部與三頭肌)

此文件概述了最佳的「推日」（Push-day）訓練策略，針對胸部（胸大肌 / Pectorals）、前側和外側肩部（三角肌 / Deltoids）以及三頭肌（Triceps）。它專注於符合生物力學的動作選擇、漸進超負荷（Progressive Overload）以及以拉伸為核心的訓練。

---

## 1. 核心動作與動作替換 (Cornerstone Movements & Exercise Swaps)

為了建立強壯且健康的上半身，優先選擇能最大化肌肉張力且能保護關節和旋轉肌群 (Rotator Cuff) 的動作。

### 上半身推的核心動作 (Upper Body Push Cornerstones)
1. **垂直推 (Vertical Push)**：**坐姿啞鈴過頂推舉** (Seated Dumbbell Overhead Press)
   * *主要肌肉*：前三角肌 (Anterior Deltoid)、側三角肌 (Lateral Deltoid)、上胸部 (Upper Chest)。
   * *原理*：坐姿推舉優於站姿，因為它減少了對身體穩定性 (Stability Demands) 的要求。這允許承載更重的負荷，並產生大約 **10% 更高的力量** 輸出。
2. **水平推 (Horizontal Push)**：**平板槓鈴臥推** (Flat Barbell Bench Press，或啞鈴臥推 / Dumbbell Bench Press)
   * *主要肌肉*：胸大肌（胸部 / Pectorals）、前三角肌、三頭肌。
   * *原理*：臥推強度與胸部肥大（增肌）之間存在直接且強烈的正相關。如果槓鈴導致肩部不適，可以使用啞鈴，因為啞鈴能提供更大的運動範圍（ROM）和更好的關節舒適度。

### 生物力學推動作替換 (Biomechanical Push Swaps)

將常見且限制關節結構的動作替換為能在拉伸狀態下提供負載的變式：

| 較差的動作 | 生物力學限制 | 更優的替代方案 | 原理 |
| :--- | :--- | :--- | :--- |
| **標準俯臥撐 (Standard Push-Up)** | 由於地面的限制，底部胸部拉伸有限；難以增加負重。 | **赤字俯臥撐 (Deficit Push-Up)** | 使用俯臥撐架或墊木下沉至手部水平線以下，創造深度的胸部拉伸。 |
| **啞鈴三頭俯身臂屈伸 (Dumbbell Tricep Kickback)** | 在拉伸位置零張力；只有在頂端收縮時最難。 | **過頂繩索臂屈伸** (Overhead Cable Extension) | 在恆定的繩索張力下，最大化三頭肌長頭 (Long Head) 的被動拉伸。 |
| **上斜俯臥撐 (Incline Push-Up)** | 自重變式，運動範圍有限。 | **上斜啞鈴臥推 (Incline DB Press) / 赤字俯臥撐** | 更好的負荷自我調節，並為上胸部提供更深的機械拉伸。 |

## 2. 胸部訓練 (以下胸為重點 - Lower Pecs Focus)

為了建立完整飽滿的下胸部，專注於模擬下胸纖維向下掃掠軌跡的動作。下斜臥推（Decline Pressing）並非必要；平板推和飛鳥/交叉拉伸動作更加優越。

> [!TIP]
> 下胸的發展不需要下斜臥推。理解胸部解剖結構與功能，使你能夠利用平板夾角和由高到低的掃掠軌跡來孤立下胸。

### 建議的胸部動作 (Suggested Chest Exercises)

#### 1. 平板臥推 (Flat Bench Press - 槓鈴或啞鈴)
* **準備動作**：收緊並下沉你的肩胛骨 (Scapula)，保持下背部的自然微拱，雙腳踩實地面。
* **動作執行**：使用中寬握距。有控制地下放槓鈴至中胸部，在底部拉伸位置稍微停頓，然後推起。
* **容量**：3–4組 (Sets)，每組6–8次 (Reps)。

#### 2. 仰臥拉起 (Bench Pullovers)
* **準備動作**：垂直躺在平板凳上，僅將上背部靠在長凳上。臀部下沉至長凳水平線以下以增加拉伸感。
* **動作執行**：雙手抱握啞鈴過頂。微微屈肘將其向後下放，完全專注於拉伸感。將其拉回至額頭上方（不要帶到胸部上方，否則會失去張力）。
* **容量**：3組，每組10–12次。

#### 3. 由高到低繩索交叉飛鳥 (High-to-Low Cable Crossovers)
* **準備動作**：將繩索滑輪設置到最高位置。呈前後弓步站立以保持穩定，保持軀幹挺直。
* **動作執行**：將手柄向下向內拉合至腰部高度。專注於向下的掃掠動作。
* **高級變式**：在底部雙手稍微交叉，以最大化下胸的收縮。
* **容量**：3組，每組12–15次。

### 建議的胸部訓練結構 (Suggested Chest Day Structure)

| 動作 | 組數 | 次數 | 重點 |
| :--- | :---: | :---: | :--- |
| **平板臥推** | 3–4 | 6–8 | 整體胸部維度的主要複合動作。 |
| **仰臥拉起** | 3 | 10–12 | 通過肩關節伸展（Shoulder Extension）拉伸來針對下胸。 |
| **高到低繩索飛鳥** | 3 | 12–15 | 底部收縮張力極高的收尾動作。 |

## 3. 肩部推舉與側平舉 (Shoulder Pressing & Lateral Raises - 前側與側三角肌)

三角肌是高度分區的。為了建立寬闊的「南瓜肩」（Boulder Shoulders），優先訓練側三角肌 (Lateral Delts)。前三角肌 (Anterior Delts) 在胸部推舉過程中已經獲得了充足的刺激。

> [!WARNING]
> 避免在肩部熱身時進行過度牽拉的動作，例如將PVC管繞至背後。這會給關節唇 (Labrum) 和旋轉肌群 (Rotator Cuff) 帶來壓力。保持正常的肩胛骨運動，避免過度拉伸。

### 肩部熱身：活動度 (Shoulder Warm-up: Mobility)
* **重點**：強調肩胛骨-肱骨節律 (Scapulohumeral Rhythm)。避免過大的運動範圍以防止肩關節撞擊 (Impingement)。用輕重量的外展/外旋動作熱身旋轉肌群。

### 肩部動作 (Shoulder Exercises)

#### 1. 繩索側平舉 (Cable Side Lateral)
* **動作執行**：將繩索滑輪設置在手腕高度。側對繩索機站立，將繩索向側上方拉起。專注於手腕的**內旋** (Internal Rotation，小拇指略高於大拇指)，以針對側三角肌中容易被忽視的肌纖維。
* **執行技巧**：避免拉高過平行線（肩膀高度），以防斜方肌 (Trapezius) 主動發力。
* **增肌強化**：專注於「向外延伸」而不是「向上拉起」。

#### 2. 史密斯機寬距直立划船 (Smith Machine Wide Grip Upright Row)
* **動作執行**：寬握槓鈴。沿著胸部向上拉，保持手肘高於槓鈴，以收縮側三角肌。專注於肩部的內旋。

#### 3. 前三角肌推舉 (Tucked-Elbow Shoulder Press，肘內收肩推)
* **動作執行**：進行坐姿啞鈴推舉，手肘稍微向前收攏（在肩胛骨平面 / Scapular Plane 上，約45度角）而不是向外大張。這能更好地孤立前三角肌並保護肩關節。

#### 4. 簡易陸上地雷架推舉 (Makeshift Viking Press)
* **動作執行**：使用槓鈴陸上地雷（Landmine）設置。採用中性握姿向上推舉。這是一個極佳的、對關節友好的推舉動作，適合作為前三角肌的收尾動作。

## 4. 三頭肌訓練 (Triceps Training)

三頭肌訓練應該針對長頭 (Long Head，需要過頂拉伸)、側頭 (Lateral Head) 和內側頭 (Medial Head)。

* **訓練頻率 (Frequency)**: 每週1–2次，間隔至少48小時恢復。
* **每組間休息 (Rest)**: 60–90秒。
* **訓練強度 (Intensity)**: 最後一組推至力竭 (Failure) 後，繼續進行拉伸部分重複 (Lengthened Partials) 動作。

### 三頭肌訓練結構 (Triceps Workout Structure)

#### 1. 上拉繩三頭肌伸展 (Overhead Cable Extension)
* **組數**：3–4組
* **次數**：10–12次
* **目標肌肉 (Target)**：三頭肌的長頭（後方，在伸展位置進行高負荷訓練 / Loaded in the Stretched Position）
* ![Tricep overhead extension](../attach/Pasted%20image%2020241107121333.png)

#### 2. 組合動作：單手拉繩下拉 + 橫跨身體三頭肌下拉 (Combo: Single-Arm Pull Down + Across-Body Pull Down)
* **組數**：每隻手3組
* **次數**：每個動作10–12次
* **目標肌肉 (Target)**：主要針對三頭肌的側頭（寬度 / Width）和內側頭（中央形狀 / Medial and Lateral Heads）
* *參考影片*: [參考影片 (1st combo)](https://www.youtube.com/watch?v=waku-nPjAM8)

#### 3. 窄握槓鈴臥推 (Close-Grip Barbell Press)
* **組數**：3–4組
* **次數**：6–8次
* **目標肌肉 (Target)**：三頭肌的內側頭和側頭（增強厚度與力量），次要啟動胸部肌肉
