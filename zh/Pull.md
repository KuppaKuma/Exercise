# 🟢 Pull 訓練指南 (背部、後三角肌、二頭肌與腹肌)

此文件概述了最佳的「拉日」（Pull-day）訓練策略，針對背部（背闊肌、斜方肌和菱形肌）、後肩（後三角肌）、二頭肌（Biceps）和核心（腹肌 / Abs）。它專注於基於科學的動作選擇、正確的拉起軌跡以及完全的機械拉伸（Mechanical Stretch）。

---

## 1. 核心動作與動作替換 (Cornerstone Movements & Exercise Swaps)

為了建立背部的寬度與厚度，優先選擇能提供完整運動範圍（Range of Motion / ROM）且能最大程度減少下背部疲勞的動作。

### 上半身拉的核心動作 (Upper Body Pull Cornerstones)
1. **垂直拉 (Vertical Pull)**：**寬距引體向上** (Wide-Grip Pull-Up)
   * *主要肌肉*：背闊肌 (Latissimus Dorsi / Lats)、大圓肌 (Teres Major)、二頭肌 (Biceps)、前臂 (Forearms)。
   * *原理*：較寬的握距能最大化背闊肌的激活。引體向上對核心肌群的徵召顯著優於高位下拉 (Lat Pulldowns)。
2. **水平划船 (Horizontal Row)**：**胸部支撐划船** (Chest-Supported Row)
   * *主要肌肉*：斜方肌 (Trapezius / Traps)、菱形肌 (Rhomboids)、後三角肌 (Rear Delts)、背闊肌 (Lats)。
   * *原理*：消除下背部疲勞和借力（Momentum），確保嚴格孤立上背部。手肘外展可將重點轉移到上背部和斜方肌。

### 生物力學拉動作替換 (Biomechanical Pull Swaps)

將常見、容易導致疲勞的動作替換為具有更好生物力學和張力曲線的變式：

| 較差的動作 | 生物力學限制 | 更優的替代方案 | 原理 |
| :--- | :--- | :--- | :--- |
| **俯身划船 (Bent Over Row)** | 限制運動範圍；下背部和姿勢維持疲勞度高。 | **胸部支撐 T-Bar 划船 / 史密斯划船** (Chest-Supported T-Bar / Smith Row) | 消除下背部壓力，在底部提供完全的拉伸。 |
| **俯身後三角肌飛鳥 (Bent Over Rear Delt Fly)** | 導致姿勢性下背部疲勞；張力曲線差。 | **繩索交叉後三角飛鳥** (Rear Delt Cable Crossover) | 繩索提供恆定張力；孤立後三角肌且無下背部負擔。 |

## 2. 背部與背闊肌訓練 (Back & Lats Training)

為了建立寬闊、厚實的背部（V字倒三角 / V-taper），安排你的訓練以同時針對中下/上背闊肌和中背部（斜方肌與菱形肌）。

> [!TIP]
> 始終用「手肘」帶動發力拉起，而不是用手掌抓拉。專注於意念肌肉連結（Mind-Muscle Connection），並控制每次反覆動作的離心（下放）階段（Eccentric Phase）。

### 動作指南 (Exercise Guides)

#### 1. 高位下拉 (Lat Pulldowns，針對下背闊肌)
* **動作執行**：使用與肩同寬的對稱握法（中性握姿 / Neutral Grip）或反握（Underhand Grip）。用手肘向下、向臀部方向拉成一條微小的弧線。保持肩膀下沉（Depressed，遠離耳朵），並在頂部完全拉伸。
* **增肌強化 (Hypertrophy Boost)**：在最後一組的拉伸位置進行半程重複動作（拉伸部分重複 / Lengthened Partials）。
* ![Lat Pulldown](../attach/Pasted%20image%2020240826133644.png)

#### 2. 繩索划船或啞鈴划船 (Cable/Dumbbell Rows，針對上背闊肌)
* **動作執行**：採用中性握姿，雙手與肩同寬。身體微微前傾，收緊核心（Core）。將手肘向後向下收縮至臀部位置。讓肩膀在底部向前充分伸展（Stretch）。
* ![Cable Row / Dumbbell Row](../attach/Pasted%20image%2020240826134247.png)

#### 3. 寬距划船 (Wide Grip Rows，針對中背部和上斜方肌)
* **動作執行**：在高位下拉桿或划船機上使用寬距正握（Overhand Grip）。身體微微後仰，將槓下拉至下胸部，同時手肘外展（Flaring Elbows Out），並擠壓兩側肩胛骨。

### 背部動作優先級列表 (Back Exercise Priority List)
按肌電圖 (EMG) 肌肉激活度和漸進超負荷（Progressive Overload）實用度排序：
1. **正手/反手引體向上 (Pull-Ups / Chin-Ups，高優先級)**：對背闊肌、斜方肌和二頭肌有著無可比擬的激活。
2. **啞鈴仰臥拉起 (Dumbbell Pullover，高優先級)**：連接胸部與背部；在負重下拉伸效果極佳。
3. **胸部支撐啞鈴划船 (Chest-Supported Dumbbell Row，高優先級)**：動作規範嚴格，零下背部疲勞。
4. **米多斯划船 (Meadows Row，中優先級)**：針對後三角肌、斜方肌和前臂的單側（Unilateral）槓鈴划船。
5. **寬距低位划船 (Wide Grip Pulley Row，中優先級)**：增強上背部寬度和上背闊肌。
6. **仰臥懸垂划船 (Inverted Row，中優先級)**：紮實的自重水平拉動作。
7. **拖雪橇 (Sled Pull，低優先級)**：適合體能訓練（Conditioning），但對單純的肌肉肥大（Hypertrophy）並非最佳。

### 建議的背部訓練結構 (Suggested Back Day Structure)

1. **高位下拉**：3–4組 (Sets)，每組8–12次 (Reps)。專注於下背闊肌的收縮；在最後一組加入拉伸位置的半程重複。
2. **繩索划船或啞鈴划船**：3–4組，每組8–12次。強調每次反覆底部的拉伸。
3. **寬距划船**：3–4組，每組8–12次（繩索機或T-bar）。
   * **最後一組特別斜方肌降組技術 (Special Trap Dropset Technique)**:
     1. 完成全程重複次數（Full Reps）至力竭（Failure）。
     2. 完成半程（拉伸位置）重複次數至力竭。
     3. 進行肩胛骨擠壓（聳肩 / Shrugs）至力竭。

## 3. 後三角肌訓練 (Rear Delts Training - Posterior Head)

由於斜方肌和背闊肌等較強肌肉的干擾，後三角肌往往容易被忽視或訓練效果不佳。

### 後三角肌激活機制 (Rear Delt Activation Mechanics)
* **斜方肌干擾 (Trap Interference)**：標準的後三角肌動作常常失敗，因為斜方肌和菱形肌會主導發力。
* **解決方法 (The Fix)**：保持手臂位置較低、靠近身體（不要呈90度外展），並專注於「勺狀」動作（Scooping Motion，向外延伸並向後掃起）以孤立後三角肌。

### 後三角肌動作分級表 (Rear Delt Exercise Tier List)

* **S-Tier (最佳)**：
  * **上斜凳啞鈴後三角舉 (Incline Dumbbell Rear Delt Raise)**：俯臥在30度斜凳上，手臂靠近身體，向後作勺狀掃起。
  * **雙繩索後三角拉伸 (Dual Cable Rear Delt Raise)**：交叉繩索，手臂靠近身體，向後向外拉，維持恆定張力。
  * **單臂後三角繩索半程拉 (Single-Arm Rear Delt Cable Partial)**：單側繩索拉伸，重度專注於底部的拉伸伸展（Stretched）位置。
* **B-Tier (良好)**：
  * **側坐蝴蝶機後三角飛鳥 (Sideways Machine Rear Delt Fly)**：側坐在蝴蝶機（Pec Deck）上進行單側飛鳥，延伸後三角肌。
  * **啞鈴後三角飛鳥 (Dumbbell Rear Delt Fly - 縮短 ROM)**：站姿俯身或在斜凳上，但稍微縮短運動範圍以保持肌肉張力。
* **C-Tier (一般)**：
  * **標準啞鈴後三角飛鳥 (Standard Dumbbell Rear Delt Fly)**：難以孤立；在動作頂端斜方肌會主導發力。
* **D/F-Tier (應避免)**：
  * **俯身單側繩索飛鳥 (Bent-Over Unilateral Cable Fly - D+)**：效果有限。
  * **高位雙繩索後三角飛鳥 (High Dual Cable Rear Delt Fly - F-Tier)**：徵召了過多的斜方肌和上背部。

## 4. 二頭肌訓練 (Biceps Training)

此訓練計劃針對二頭肌的所有部位，包含 **背向拉繩二頭彎舉**（Face-Away Cable Curl）、**牧師捲舉超級組**（Preacher Curl Superset）、**槌式捲舉**（Hammer Curl）和**反向捲舉**（Reverse Curl）。每週訓練兩次，間隔1至2天休息。

* **每週總計組數 (Weekly Set Volume)**: 20組
* **訓練頻率 (Frequency)**: 每週訓練兩次，每次間隔1至2天

### 二頭肌訓練結構 (Biceps Workout Structure)

#### 1. 背向拉繩二頭彎舉 (Face-Away Cable Curl)
* **組數 (Sets)**: 3組
* **次數 (Reps)**: 8–12次
* **休息 (Rest)**: 每組間隔60–90秒
* **重點 (Focus)**: 將二頭肌置於身體後方的完全伸展位置（Behind-the-Body Stretched Position）。
* ![Face-Away Cable Curl](../attach/Pasted%20image%2020241105120424.png)

#### 2. 牧師捲舉超級組 (Preacher Curl Superset，或可用機械彎舉 / Machine Curl 替代)
* **組數 (Sets)**: 3組超級組 (Supersets)
  * **重組 (Heavy Set)**: 8–12次（較重的重量 / Heavy Weight）
  * **降組 (Drop Set)**: 8–12次（重量減半 / Half Weight）
* **休息 (Rest)**: 每組超級組結束後休息90秒
* ![Preacher Curl Superset](../attach/Pasted%20image%2020241105120450.png)

#### 3. 槌式捲舉 (Hammer Curl，啞鈴或牧師凳上進行)
* **組數 (Sets)**: 2組
* **次數 (Reps)**: 10–12次
* **休息 (Rest)**: 每組間隔60–90秒
* **重點 (Focus)**: 針對肱肌 (Brachialis) 與肱橈肌 (Brachioradialis)，增加前臂與外側手臂厚度。
* ![Hammer Curl](../attach/Pasted%20image%2020241105120556.png)

#### 4. 反向捲舉 (Reverse Curl，EZ槓或啞鈴)
* **組數 (Sets)**: 2組
* **次數 (Reps)**: 10–12次
* **休息 (Rest)**: 每組間隔60–90秒
* **重點 (Focus)**: 針對肱橈肌 (Brachioradialis) 與手腕伸肌 (Wrist Extensors)。
* ![Reverse Curl](../attach/Pasted%20image%2020241105120625.png)

### 重要提示 (Important Biceps Tips)
1. **控制下放動作和局部重複 (Controlled Negatives & Partials)**：每次動作的下放（離心 / Eccentric Phase）階段要保持控制。當無法完成完整動作時，在伸展位置（Lengthened Partials）進行 3–5 次局部重複以徹底疲勞肌肉。
2. **血流限制訓練 (Blood Flow Restriction / BFR)**：在手臂訓練時使用塑膠彈性帶進行血流限制（BFR）訓練。訓練結束後約 5 分鐘後取下彈性帶以恢復循環。

## 5. 核心訓練 (Core Training - Abs Tier List)

為了練出明顯、厚實的腹肌塊，需要在負重（阻力）下通過完整的脊柱屈曲 (Spinal Flexion) 和旋轉 (Rotation) 來訓練腹肌。

* **S-Tier (最佳)**：
  * **繩索跪姿捲腹 (Cable Crunch)**：在恆定張力下進行負重脊柱屈曲。
  * **繩索轉體 (Cable Twist)**：負重軀幹旋轉 (Loaded Trunk Rotation)。
* **A-Tier (極佳)**：
  * **下斜板仰臥起坐 (Decline Situps)**：通過屈曲提供高張力。
  * **健腹輪 (Ab Roller)**：極佳的離心負載 (Eccentric Loading)。
  * **龍旗 (Dragon Flag)**：高難度的自重核心穩定性動作。
  * **懸垂 / 羅馬椅抬腿 (Hanging / Roman Chair Leg Raises)**：非常適合下腹部。
  * **俄羅斯轉體 (Russian Twist)**：經典的腹外斜肌（Obliques）旋轉動作。
* **B-Tier (一般)**：
  * **經典仰臥起坐 (Classic Situps)** / **倒掛仰臥起坐 (Inverted Situps)**。
* **C-Tier (較差)**：
  * **平板支撐 (Planks)**：靜態等長收縮（Static Isometric Holds）與動態負重屈曲相比，無法帶來顯著的肌肉肥大。
* **D-Tier (次佳)**：
  * **登山跑 (Mountain Climbers)**、**啞鈴體側屈 (Dumbbell Side Bends)**（可能會無謂地加寬腰圍）。
* **避免 (無效)**：
  * **基礎仰臥起坐 (Crunches)** / **單車捲腹 (Bicycle Crunches)**（運動範圍有限，無阻力）。
